# 대구교육데이터 시각화 대시보드(시범)

이 프로젝트는 대구교육데이터를 시각화하여 보여주는 대시보드 갤러리입니다.

## 주요 기능

- 대시보드 갤러리 형태의 메인 페이지
- Tableau 대시보드 임베딩
- 반응형 디자인
- 모던한 UI/UX

## 기술 스택

- Next.js
- TypeScript
- Tailwind CSS
- GitHub Pages

## 로컬 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build
```

## 배포

이 프로젝트는 GitHub Pages를 통해 자동으로 배포됩니다. `main` 브랜치에 푸시하면 GitHub Actions를 통해 자동으로 배포가 진행됩니다.
